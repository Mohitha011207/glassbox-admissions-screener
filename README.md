## 🚀 Overview

Glass Box Admissions Screener is a modern AI-powered system designed to bring **transparency, fairness, and explainability** into student admissions.

Unlike traditional black-box systems, this application:
- Clearly explains *why* a student is accepted or rejected  
- Detects potential bias in decision-making  
- Provides actionable insights for improvement  

---

## 🎯 Problem Statement

Most AI-based admission systems operate as **black boxes**, leading to:
- Lack of transparency  
- Algorithmic bias  
- Unfair decision-making  

Students are often rejected without understanding *why*.

---

## 💡 Our Solution

Glass Box introduces a **transparent AI model** that:
- Shows decision logic  
- Builds trust with users  
- Encourages data-driven improvement  

---

## ✨ Key Features

### 📂 CSV Upload & Data Processing
- Upload student datasets بسهولة
- Automatic validation and parsing

### 🤖 Prediction Engine
- Predicts **Admit / Reject**
- Generates **confidence score**

### 🔍 Explainable AI
- Shows key factors influencing decisions
- Highlights feature impact (GPA, internships, etc.)

### ⚖️ Bias Analysis
- Detects potential bias in dataset
- Visualizes fairness across groups

### 🔁 What-If Simulator
- Modify inputs like GPA or projects
- Instantly see outcome changes
- Generates **improvement roadmap**

### 📊 Insights Dashboard
- Overall system insights
- Most influential features
- Fairness indicators

## 🧠 Working Principle

1. User uploads dataset (CSV)
2. Data is processed and validated
3. Prediction logic evaluates student profiles
4. Results generated with confidence score
5. Explanation layer highlights key factors
6. Bias analysis checks fairness
7. Simulator allows scenario testing
